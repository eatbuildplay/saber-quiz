Saber Quiz is a quiz system designed for Elementor.

== Description ==

Saber Quiz provides a powerful quiz system for use with Elementor page builder. It does not require the Elementor Pro version. Quizzes are stored as custom posts in the WP admin, and there is a separate question pool for managing questions. The user-friendly interface provide an ajax-powered quiz editor where you can add questions without page refreshes. Saber Quiz is related to Saber LMS and provides the same testing engine capacity as the full-scale Saber LMS. Saber Quiz is designed for sites that want a testing engine, without having to install a full LMS system.

== Frequently Asked Questions ==

= What requirements are there to install Saber Quiz? =

You should have WP 5.x+. Saber Quiz has been tested primarily on WP5.5. You must also have the Elementor page builder installed, minimum version 2.9.3.

= Are quiz statistics stored? =

Yes, Saber Quiz has a post meta based statistics storage approach. You will be able to see a list of all quiz attempts and scoring data associated to each user/student.
